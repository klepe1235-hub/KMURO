# Karina Muro Digital Teacher Portfolio

This repository hosts Karina Muro's digital teacher portfolio using GitHub Pages.

## GitHub Pages setup

1. Make sure `index.html` is in the root of the repository.
2. Go to Settings > Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Select Branch: `main` and Folder: `/root`.
5. Save and wait a few minutes.

The live site should open at:
https://klepe1235-hub.github.io/KMURO/
