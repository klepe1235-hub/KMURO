# Karina Muro Digital Teacher Portfolio

This repository contains a self-contained GitHub Pages portfolio website.

## Deploy on GitHub Pages

1. Upload `index.html` and `README.md` to the root of the repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/root`, then save.

The portfolio is self-contained, so no separate assets folder is required.
